Add Simple Intro Boxes

To add simple intros to your site you only need to use data-intro. Add data-intro to any HTML elements that you would like to highlight for instructions or an introduction. Whatever value you include in the quotations will appear in the intro box for your user to read. For example, if I wanted to instruct a user on how to get to the next section I could include the following in my HTML link to the "Steps" section:

<a href="steps.html" data-intro="Click here to get to the next section.">Steps</a>

To get your box to show up you simply call the JavaScript function introJs().start(). I've added an instruction to the following button, for example, to call the introJs().start() function when clicked. To see the function called and the intro box added to my page above click the Example button below. You can choose to call the function however you like, but I started with a simple button with onclick="introJs().start()" added to it. The only JavaScript that needs to be added to your page is a call to the function:

<script src="intro.js"> introJs().start(); </script>

It's so simple! Go whip up a bunch of your own intros or see what else you can do by visiting the next section.
