Number and Navigate Around Your Steps

Intro.js will automatically number and navigate your intro steps from top to bottom, but you have the option to set the number, or priority, of your steps to whatever you would like. Use data-step along with your data-intro, but enter the number you would like to set for that step. For an example click the Example button below:

Example code:

<h2 data-intro="The step for this intro has been set to '1'" data-step="1">

Navigate

There are a few different functions you can use to navigate around your different steps. introJs.goToStep(step) for example will go to whichever step you input as the parameter. If I wanted to start the user at step #3 I would use the following function:

introJs().gotostep(3).start()

The function introJs.nextStep() will go to the next step of the intro. Including the following will start the user at step #2:

introJs().start().nextStep()

The function introJs.previousStep() will go to the previous step of the intro. Including the following will start the user at step #1:

introJs().goToStep(2).previousStep()
