Welcome to the how to guide for the JavaScript library Intro.js

Intro.js ads simple introductions to your website. You can use them to introduce features of your website or as a step-by-step guide for visitors. Click the button below to see an example:

Getting Started

To get started using Intro.js in your own website, you first need to get the source files. Download the latest versions from github here.

To use it in your web page simply include the intro.js and introjs.css in your html code. It will look something like this (be sure to include the correct file paths):

<script src="intro.js"></script>
<link rel="stylesheet" type="text/css" href="introjs.css"/>
